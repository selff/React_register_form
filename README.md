# React_register_form

## Overview

This is a simple register form on React 
with 
- international Phone field 
- autocomplete field for Profession

## Install

```
$ git clone https://github.com/selff/React_register_form.git
$ cd React_register_form
$ npm install
```

## Start

```
$ npm start
```
